# Utility Projects

A collection of utility web applications built with Angular.

## Projects

### 1. [PDF Tool](./pdftool/)
A PDF utility tool for viewing, merging, splitting, and manipulating PDF files.

### 2. [Interview Helper](./interview-helper/)
An AI-powered interview helper application for managing candidates, screening resumes, conducting interviews, and generating reports.

## Getting Started

Each project is self-contained with its own `package.json`. Navigate to the respective project folder and run:

```bash
npm install
ng serve
```
